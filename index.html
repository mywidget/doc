Instalasi

Ikuti langkah-langkah di bawah ini untuk menyiapkan aplikasi Anda:

  1. Buka zip paket yang diunduh dan buka folder /multi.kalkulatorcetak.go-master untuk menemukan semua file template. Anda perlu mengunggah file-file ini ke server web hosting Anda menggunakan FTP atau localhost untuk menggunakannya di situs web Anda.
  2. Di bawah ini adalah struktur folder dan perlu diunggah ke situs web atau direktori root localhost Anda:
    • class - Berisi Fungsi dan config API
      • detail - Stylesheet files
      • images - Images files
      • produk - Images Modul/Produk
      • themes - Style & Template
    • .htaccess - htaccess
    • index.php - Beranda
    • README.md - README.md
  3. You should upload all or specific HTML files as per your need.
  4. You are good to go for adding your content now!

Pengaturan

Buka file app_config.php yang berada di folder class, kemudian lakukan perubahan pada :
APPS_ID dan APPS_PUBLIC

$config = array(
	"APPS_ID"	=> "YOUR_APP_ID",
	"APPS_PUBLIC"	=> "YOUR_APPS_SECRET",
	"SITE_API"	=> "https://api.kalkulatorcetak.com"
);

Jika Anda memerlukan informasi lebih lanjut, silakan kunjungi situs kalkulatorcetak: https://kalkulatorcetak.com
